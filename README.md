# IA-04: Campus Event Management Platform — Sprint Implementation

A Python service layer implementing core event management functionality for the Crerar Campus Events Platform.

## Prerequisites

Python 3.7 or higher. To check:
```bash
python3 --version
```

## Setup

```bash
# 1. Navigate to the project directory
cd ia-04

# 2. Install dependencies
pip install -r requirements.txt
```

## Running Tests

Run all tests from the IA-04 directory:
```bash
pytest tests/ -v
```


There are further examples in docs\sprint_review.md
